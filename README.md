## Getting Started

```bash
# Install dependencies
pnpm install

# Run the development server
pnpm dev
```

## Testing

The project uses Playwright for end-to-end testing.

```bash
# Run tests
pnpm test

# Run tests with UI
pnpm test:ui

# Run tests in headed mode
pnpm test:headed
```
